# Orlix AI

**Personal AI Operating System on Base**

Orlix is a unified AI interface that runs 19 frontier models alongside real-time onchain intelligence — token analysis, wallet monitoring, and a Telegram bot — all in one tab.

---

## What Orlix Does

### 🤖 Multi-Model AI Playground
Run any of 19 AI models in a single chat interface without switching tabs or losing context.

Supported models:
- **Anthropic**: Claude Sonnet, Claude Haiku, Claude Opus
- **OpenAI**: GPT-4o, GPT-4o Mini, o1, o3, o4-mini
- **xAI**: Grok 3, Grok 3 Mini
- **Google**: Gemini 2.0, Gemini 2.5 Pro, Gemini Flash
- **DeepSeek**: DeepSeek V3, DeepSeek R1
- **Groq**: Llama 3, Mixtral (ultra-fast inference)
- **Mimo**: Mimo models

### 🔍 Token Analyzer
Paste any Base contract address and get a full AI-powered risk report in seconds.

What you get:
- Live price, market cap, FDV, liquidity
- Buy/sell pressure bars
- 1h / 6h / 24h price changes and volume
- Liquidity/MCap ratio
- AI verdict: Safe / Caution / High Risk

Data sources: DexScreener API + Base RPC (real-time, no cache)

### 📱 Telegram Bot
Full Orlix access inside Telegram via **@orlixai_bot**.

- All 19 AI models
- Token analysis on any Base address
- No app needed
